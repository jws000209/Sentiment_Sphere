# AGENTS.md

## 프로젝트 목적
HTML, CSS, JS 기반 감성 분석 서비스. OpenAI로 긍정/부정/중립, 신뢰도, 이유, 오류를 표시.

## 기술스택
- Front: HTML/CSS/Vanilla JS
- Back: Node.js + Express
- AI: OpenAI
- DB: Supabase
- Deploy: Vercel

## 규칙
- 명세 외 기능 추가 금지
- API Key는 백엔드 .env만 사용
- 반응형 구현
- Semantic HTML, ES6 사용

## 완료 기준
- 요구사항 충족
- Console Error 없음
- 반응형 정상
